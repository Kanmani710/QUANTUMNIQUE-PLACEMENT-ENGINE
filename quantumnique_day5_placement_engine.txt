BLOCK 4 – PROBLEM SOLVING (Intermediate)

Find first repeating element in array

Input:
[1,2,3,4,2,5,3]

Output:
2

Follow-up:
Solve efficiently using hashing.
Solution:
import java.util.*;
public class Main {
    public static void main(String[]args) {
        Scanner sc=new Scanner(System.in);
        int n=sc.nextInt();
        int arr[]=new int[n];
        for(int i=0;i<n;i++) {
            arr[i]=sc.nextInt();
        }
        for(int i=0;i<n;i++) {
            for(int j=i+1;j<n;j++) {
                if(arr[i]==arr[j]) {
                    System.out.println(arr[i]);
                }
            }
        }
        System.out.println("Nothing");
    }
}

BLOCK 5 – DATA STRUCTURES & ALGORITHMS (Important Pattern)

Topic: Binary Search

Problem – Search in Sorted Array

Given a sorted array, find index of target element.

Example:

nums = [1,2,3,4,5,6,7]
target = 5

Output:
4

Python Solution

def binarySearch(nums, target):
    left, right = 0, len(nums)-1

    while left <= right:
        mid = (left + right)//2

        if nums[mid] == target:
            return mid
        elif nums[mid] < target:
            left = mid + 1
        else:
            right = mid - 1

    return -1

Time Complexity: O(log n)

Practice Problem

Find square root of a number using binary search
import java.util.*;
public class Main {
    public static void main(String[] args) {
        Scanner sc=new Scanner(System.in);
        int n=sc.nextInt();
        int l=1,h=n;
        int a=0;
        while(l<=h) {
            int m=(l+h)/2;
            if(m*m==n) {
                System.out.println(m);
                return;
            }
            else if(m*m<n){
                a=m;
                l=m+1;
            }
            else{
                h=m-1;
            }
        }
        System.out.println(a);
    }
}
