BLOCK 4 – PROBLEM SOLVING

Reverse a Number

Input
1234

Output
4321

Hint

Use:
remainder = n % 10
n = n // 10

Solution:

import java.util.Scanner ;
public class Main{
    public static void main(String[]args){
        Scanner sc=new Scanner(System.in);
        int n=sc.nextInt();
        int rev=0;
        while(n!=0){
            int digit=n%10;
            rev=rev*10+digit;
            n=n/10;
        }
        System.out.println(rev);
    }
}

BLOCK 5 – DATA STRUCTURES & ALGORITHMS

Topic: Arrays

Problem – Maximum Subarray (Kadane's Algorithm)

Given array:

[-2,1,-3,4,-1,2,1,-5,4]

Find the maximum sum of a contiguous subarray.

Expected Output
6

Subarray
[4,-1,2,1]

Python Solution

def maxSubArray(nums):

    current = nums[0]
    maximum = nums[0]

    for i in range(1,len(nums)):
        current = max(nums[i], current + nums[i])
        maximum = max(maximum, current)

    return maximum

Time Complexity
O(n)

Practice Problem

Find the second largest number in the array

[5,8,2,10,7]
solution
import java.util.*;
class Main {
    public static void main(String args[]) {
        Scanner sc = new Scanner(System.in);
        int n=sc.nextInt();
        int arr[]=new int[n];
        for(int i=0;i<n;i++) {
            arr[i]=sc.nextInt();
        }
        int one=Integer.MIN_VALUE;
        int two=Integer.MIN_VALUE;
        for(int i=0;i<n;i++) {
            if(arr[i]>one) {
                two=one;
                one=arr[i];
            }
            else if(arr[i]>two&&arr[i]!=one) {
                two=arr[i];
            }
        }
        System.out.println(two);
    }
}

BLOCK 6 – CORE CS CONCEPTS

DBMS CONCEPT – FOREIGN KEY

A Foreign Key is a column in one table that refers to the Primary Key
of another table. It creates a relationship between tables.

Example

Student Table

StudentID | Name
1         | Rahul
2         | Priya

Enrollment Table

StudentID | Course
1         | Java
2         | Python

Here StudentID in Enrollment is a Foreign Key.

Recursion occurs when a function calls itself.

    if n == 0:

INTERVIEW QUESTION OF THE DAY

What is the difference between Primary Key and Foreign Key?

Primary Key
- Uniquely identifies each row
- Cannot be NULL
- One per table

Foreign Key
- References primary key in another table
- Can have duplicates
- Multiple allowed
